# Pustak 📚

Pustak is a modern web-based project designed to showcase a clean and responsive interface for browsing books or educational resources. The project focuses on simplicity, responsive layout, and an intuitive user experience.

## 🚀 Live Demo

https://ashucj17.github.io/pustak/

## 📌 Features

- Responsive website design
- Clean UI for book/education platform
- Easy navigation layout
- Optimized for desktop and mobile
- Lightweight and fast loading

## 🛠️ Tech Stack

- HTML5
- CSS3
- JavaScript
- Git & GitHub
- GitHub Pages (Deployment)

## 💻 Installation

Clone the repository


git clone https://github.com/ashucj17/pustak.git


Navigate to the project folder


cd pustak


Run the project

Open `index.html` in your browser.

## 🌐 Deployment

This project is deployed using **GitHub Pages**.

Steps to deploy:

1. Go to **Repository Settings**
2. Open **Pages**
3. Select **Deploy from branch**
4. Choose branch **main**
5. Select folder **/root**
6. Save the settings

Your site will be live at:


https://ashucj17.github.io/pustak/
